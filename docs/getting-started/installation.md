# インストール

このページはインストール手順のサンプルです。

## 前提条件

*   Node.js (バージョン X.Y.Z 以上)
*   Bun (バージョン A.B.C 以上)

## 手順

1.  リポジトリをクローンします。
    ```bash
    git clone https://github.com/tKwbr999/adk-docs-ja.git
    cd adk-docs-ja
    ```
2.  依存関係をインストールします。
    ```bash
    bun install
    ```
3.  開発サーバーを起動します。
    ```bash
    bun run dev
    ```

これで `http://localhost:5173` (または指定されたポート) でサイトが表示されます。
